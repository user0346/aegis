AEGIS - Installation
=====================

REQUIREMENTS
------------
- Windows 10 or 11 (64-bit)
- No Python, no admin rights required. Everything is in the app.

INSTALL
-------
1.  Move the entire "AEGIS" folder to a permanent location,
    e.g.  C:\Program Files\AEGIS\  or  C:\AEGIS\
    (avoid Desktop or Downloads — keep the whole folder together).

2.  Double-click  AEGIS.exe.
    The interface opens and background protection starts automatically.

3.  Open  Settings -> System  and click  "First-time setup".
    This adds autostart at login, a desktop/start-menu shortcut, and the
    integrity baseline. (One click — no console, no scripts.)

That's it. AEGIS protects in the background and starts with Windows.

USE - everything is a button in the app
---------------------------------------
Settings -> System card:
  - First-time setup        (autostart + shortcut + baseline)
  - Autostart ON / OFF
  - Re-pin integrity        (after a deliberate update)
  - Restart                 (clean restart of service + interface)

Close the window any time — protection keeps running in the tray.
Right-click the tray icon to reopen, hide, or fully stop protection.

OPTIONAL FEATURES (off by default)
----------------------------------
- Local AI (Ollama): enable in the app -> Settings. Runs fully local.
- Voice control:      enable in the app -> Voice tab.
No account or API key required for either.

SECURITY PIN
------------
Set an Owner-PIN in Settings to protect autonomous actions and to require
the PIN before protection can be stopped.

UPDATES
-------
New versions are shown in the app and applied after you confirm them
(downloaded and signature-verified via cosign / Sigstore).

UNINSTALL
---------
1.  Settings -> System -> Autostart OFF, then tray -> "Stop protection".
2.  Delete the AEGIS folder. Remove the desktop/start-menu shortcut.
