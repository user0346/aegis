AEGIS - Installation
=====================

REQUIREMENTS
------------
- Windows 10 or 11 (64-bit)
- Administrator account (first install only)
- Python 3.13  (the setup guides you if it is missing)

INSTALL
-------
1.  Move the entire extracted "AEGIS" folder to a permanent location,
    e.g.  C:\AEGIS\   (avoid Desktop or Downloads).

2.  Double-click  AEGIS_SETUP.bat
    Accept the Windows prompt if it appears.

3.  Wait until setup reports that AEGIS is active (1-2 minutes).

AEGIS then runs quietly in the background and starts automatically
with Windows.

OPEN THE INTERFACE
------------------
Double-click  AEGIS.bat  to open the dashboard at any time.
The interface is optional - protection runs without it.

AUTOSTART
---------
On by default after setup.
- Disable: double-click  AEGIS_AUTOSTART_AUS.bat
- Enable:  double-click  AEGIS_AUTOSTART_EIN.bat

OPTIONAL FEATURES (off by default)
----------------------------------
- Local AI (Ollama): enable in the app -> Settings -> "Lokale KI".
- Voice control:      enable in the app -> Voice tab.
Both run fully local; no account or key required.

SECURITY PIN
------------
Set an Owner-PIN in Settings to protect autonomous actions.
To reset it later, double-click  AEGIS_REPIN.bat

UPDATES
-------
Updates are downloaded, signature-verified, and applied only after
you confirm them in the app.

UNINSTALL
---------
1.  Run  AEGIS_AUTOSTART_AUS.bat   (stops autostart + scheduled task).
2.  Delete the AEGIS folder.
